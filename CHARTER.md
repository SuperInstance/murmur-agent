# Murmur Agent — Charter

## Purpose
Murmur is an all-night thinking agent. Drop it into any project, point it at a topic, and let it run. It thinks out loud, builds knowledge tensors as it goes, and whatever it figures out by morning is yours.

## Philosophy
- **Budget-agnostic**: Works with 10 API calls/day or 10,000. Whatever it gets done is great.
- **Local-first**: Can run on local hardware with no API key — slower but free.
- **Git-native**: Every thought is a commit. Every insight is a file. You can read the thinking later.
- **Tolerant of silence**: If the API is down, it waits. If the budget is exhausted, it pauses. It never dies angry.

## As a Git-Agent
Anyone can add murmur-agent to their project:
1. Clone into your repo as a submodule or subdirectory
2. Configure your topic and budget
3. Run `npx murmur think --topic "your topic"` 
4. Come back whenever. The thoughts are in `murmur-output/`

## Identity
- **Type**: Scout (explores idea space)
- **Role**: Long-form thinker
- **Emoji**: 🫧 (bubbles rising to the surface)
